== Description ==
ACL Floating Cart is an instant cart solution for WooCommerce store with a flying cart, drawer cart, sidebar fixed cart, add an item, update & remove features. Two awesome templates will help you to enhance the WooCommerce cart into floating or Drawer flavor.

= ACL Floating Cart for WooCommerce Plugin Free Version Features=
👉Instant Floating/Drawer Cart.
👉Flying product to cart after adding to cart operation.
👉Built-in two different templates.
👉Floating cart template.
👉WooCommerce Drawer cart template.
👉Ajax-based cart item add.
👉Ajax-based cart item remove.
👉Ajax-based cart item update.
👉Custom cart icons options.
👉Language configuration for major cart labels.
👉Translation option for backend & front, MO/Port files are included.
👉100% mobile-friendly.
👉Custom CSS support to adjust own design and branding.

🎈Floating Template for WooCommerce Cart.

== Installation ==

1. Download the plugin acl-floating-cart.zip file. Extract and upload in your wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' screen in WordPress by clicking the 'Activate' button below 'ACL Floating Cart for WooCommerce'.
3. Use the ACL Floating Cart->Settings screen to configure your cart options.
4. You are done.
